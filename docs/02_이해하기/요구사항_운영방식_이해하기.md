## 핵심 개념: 역할 분리  

```
JSON 파일      →   데이터 보관창고 (하드디스크에 저장)  
Quiz 클래스    →   퀴즈 1개를 표현하는 설계도  
QuizGame 클래스 →  게임 전체를 운영하는 매니저  
```

```
JSON 파일    = 창고 (물건을 보관)  
Quiz 클래스  = 상품 박스 (물건 1개의 형태)  
QuizGame     = 가게 사장 (창고에서 꺼내고, 손님 응대하고, 다시 창고에 넣음)  
```

<br><br>

---

## 실제 흐름 그림  

```
프로그램 시작  
     ↓  
QuizGame이 state.json 파일을 읽음 (JSON 파일을 읽으면 자동으로 dict가 됩니다)  
     ↓  
JSON 데이터 → Quiz 객체들로 변환 (메모리에 올림)  
     ↓  
사용자가 게임 플레이 (Quiz 객체 사용)  
     ↓  
QuizGame이 Quiz 객체들 → JSON으로 변환해서 저장  
     ↓  
프로그램 종료  
```

## 코드로 보기  

```python
import json  

# ✅ Quiz 클래스: 퀴즈 1개를 표현  
class Quiz:  
    def __init__(self, question, choices, answer):  
        self.question = question   # 문제  
        self.choices = choices     # 선택지 리스트  
        self.answer = answer       # 정답 번호 (1~4)  

    def show(self):  
        """퀴즈를 화면에 출력"""  
        print(self.question)  
        for i, choice in enumerate(self.choices, 1):  
            print(f"{i}. {choice}")  

    def check(self, user_answer):  
        """정답 확인"""  
        return self.answer == user_answer  

    # 📦 Quiz 객체 → dict (JSON 저장용)  
    def to_dict(self):  
        return {  
            "question": self.question,  
            "choices": self.choices,  
            "answer": self.answer  
        }  
    



    # 📦 dict → Quiz 객체 (JSON 불러오기용)  
    @staticmethod  
    def from_dict(data):  
        return Quiz(data["question"], data["choices"], data["answer"])  
```

```python
# ✅ QuizGame 클래스: 게임 전체 관리  
class QuizGame:  
    def __init__(self):  
        self.quizzes = []    # Quiz 객체들의 리스트  
        self.best_score = 0  
        self.load()          # 시작하자마자 파일에서 불러옴  

    # 📂 JSON → Quiz 객체들 (불러오기)  
    def load(self):  
        try:  
            with open("state.json", "r", encoding="utf-8") as f:  
                data = json.load(f)  
                # dict 리스트를 Quiz 객체 리스트로 변환  
                self.quizzes = [Quiz.from_dict(q) for q in data["quizzes"]]  
                self.best_score = data["best_score"]  
        except FileNotFoundError:  
            # 파일 없으면 기본 데이터 사용  
            self.quizzes = self.get_default_quizzes()  
            self.best_score = 0  

    # 💾 Quiz 객체들 → JSON (저장하기)  
    def save(self):  
        data = {  
            "quizzes": [q.to_dict() for q in self.quizzes],  # 객체 → dict  
            "best_score": self.best_score  
        }  
        with open("state.json", "w", encoding="utf-8") as f:  
            json.dump(data, f, ensure_ascii=False, indent=4)  
```

## 핵심 포인트 2가지 🔑  

### 1. `dict_to_json()` - 객체를 JSON으로 저장할 때  
```python
# Quiz 객체는 JSON에 직접 저장 불가 ❌  
# dict(딕셔너리)로 변환해야 저장 가능 ✅  

quiz = Quiz("파이썬 창시자는?", ["귀도", "리누스", "제임스", "빌"], 1)  
quiz.to_dict()  
# 결과: {"question": "파이썬 창시자는?", "choices": [...], "answer": 1}  
```

### 2. `json_to_dict()` - JSON에서 객체로 불러올 때  
```python
# JSON에서 읽으면 dict로 나옴  
data = {"question": "파이썬 창시자는?", "choices": [...], "answer": 1}  

# dict → Quiz 객체로 변환  
quiz = Quiz.from_dict(data)  
quiz.show()  # 이제 객체로 사용 가능!  
```

## 전체 관계 한눈에 보기  

```
state.json (파일)  
    ↕ load() / save()  
QuizGame (매니저)  
    ↕ from_dict() / to_dict()  
Quiz 객체들 (메모리)  
    ↕ show() / check()  
사용자 (화면)  
```
